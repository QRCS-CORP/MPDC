\doxysubsection{Detailed Description}
The MPDC topology device trust structure. 

This structure contains trust-\/related information for a network device. It includes the device\textquotesingle{}s network address, domain, name, trust value, bandwidth metric, and flags to indicate IPv6 capability and whether the device is on the local network.
